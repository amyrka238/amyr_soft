# amyr - external cheat cs2 in python


# To install:

Python 3.10+
```sh
pip install requests psutil pywin32 pyautogui glfw imgui[glfw] PyOpenGL Pillow numpy scipy
```
or
```sh
pip install -r requirements.txt
```

# Launch
```sh
python px.py
```
